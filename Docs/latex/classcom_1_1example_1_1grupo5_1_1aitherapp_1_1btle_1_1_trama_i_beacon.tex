\doxysubsection{Detailed Description}
Clase que representa y descompone una trama i\+Beacon a partir de un array de bytes. 

Permite obtener el prefijo, UUID, major, minor, Tx\+Power y demás campos de la trama publicitaria recibida por BLE.
